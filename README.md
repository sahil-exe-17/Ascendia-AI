# Ascendia AI – Your Intelligent Placement Companion

Ascendia AI is a premium, AI-powered web application designed to help students excel in their placement journeys. Featuring a sleek glassmorphism UI, entrance reveal animations, and intelligent AI features.

## 🚀 Features

- **AI Chatbot**: Intelligent placement-related query handling.
- **Resume Insights**: PDF analysis, ATS scoring, and keyword suggestions.
- **Interview Simulator**: Domain-specific mock interviews with AI evaluation.
- **Career Roadmap**: Tailored 8-week path generation based on branch and role.

## 🛠️ Tech Stack

- **Frontend**: React, Framer Motion, Tailwind CSS, Lucide Icons.
- **Backend**: FastAPI (Python), Google Gemini AI.
- **Deployment**: Localhost (dev servers).

## 🏃 Getting Started

### Prerequisites
- Python 3.8+
- Node.js & npm

### Setup
1. **API Key**: Add your Google Gemini API key to `backend/.env`.
2. **Backend**:
   - Navigate to `backend`.
   - The environment is already set up (refer to `run_backend.bat`).
3. **Frontend**:
   - Navigate to `frontend`.
   - Run `npm install` (already done during creation).

### Running the App
- Run `run_backend.bat` to start the FastAPI server (Port 8000).
- Run `run_frontend.bat` to start the Vite dev server.
